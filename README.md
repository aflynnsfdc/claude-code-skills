## My Skills
There are currently 2 skills that I have added: 
- fowler-refactor
- coding-standards

Fowler refactor is based on the book 'Refactoring' by Martin Fowler. It is an attempt to encapsulate the principles in that book.

Coding standards is based on a paper called ['The Power of 10: Rules for Developing Safety-Critical Code'](https://web.eecs.umich.edu/~imarkov/10rules.pdf) by Gerard J. Holzmann. This is the basis for software written by JPL (Jet Propulsion Labs), who write the software for NASA. If it is good enough for NASA, it's good enough for me.
